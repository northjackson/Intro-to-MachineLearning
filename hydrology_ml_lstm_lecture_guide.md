# Hydrology ML Lecture Notebook Guide

This notebook is intended to accompany a graduate-level lecture on machine learning in hydrology.

It now includes:
- ML basics and evaluation
- scikit-learn, PyTorch, TensorFlow, and Keras framing
- tree-based baseline models
- neural networks
- RNNs and LSTMs
- a hydrology streamflow example

Suggested classroom flow:
1. Open with the ML ecosystem slide/cell
2. Walk through train/validation/test and scaling
3. Show the random forest baseline
4. Introduce neural networks
5. Motivate RNNs and LSTMs with hydrologic memory
6. End with the streamflow comparison plot
